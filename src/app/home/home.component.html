Przykładowa aplikacja MEAN wykonana w Angular 5

Przykładowa aplikacja MEAN wykonana w Angular 5, która może być używana jako początkowy szablon dla Twoich rzeczywistych aplikacji!

Pracę z Angular nalezy rozpocząć od instalacji odpowiedniego środowiska IDE do Angulara. Ja używam Visual Studio Code. Dane są przechowywane w bazę danych ja w tym przykładzie używam MongoDB. Server mam napisany w NodeJS. Kapilację wykonuję za pomocą nażędzia Angular CLI.

Uruchomienie projektu jest związane z instalacją wszystkich potrzebnych bibliotek. Lista znajduję się w pliku package.js. Wykonanie instlacji potrzebnych bibliotek wykonuje się poleceniem:

npm install

Podczas wykonania powyższego polecenia, należy być w katalogu głównym projektu.

Baza danych jest przechowywana w MongoDB. Object User przedstawia się następująco:
